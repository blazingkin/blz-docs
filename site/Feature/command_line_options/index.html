This page is for blz v2.6

Execute a blz file (typically .blz extension)

blz PATH [-m MAIN]

Run in immediate mode

blz -i[mmediate]

List Packages

blz -p[ackage] [help | list]

Run Test Files

blz -t[est] FILES...

Print version number

blz -v[ersion]

See this help message

blz -h[elp]
